# HireIQ - Project Description for Resume/Portfolio

## Short Description (1-2 sentences)
**HireIQ** is an AI-powered mobile application built with Flutter that helps job seekers prepare for interviews through intelligent mock interviews, resume analysis, and personalized learning experiences. The app features real-time AI conversations, PDF resume parsing, collaborative chat rooms, and cross-platform support for Android and iOS.

---

## Medium Description (Paragraph - for LinkedIn/GitHub)
**HireIQ** is a comprehensive interview preparation platform I developed using Flutter and Firebase. The application leverages AI technology (Groq API) to provide users with realistic interview practice sessions across multiple job roles and difficulty levels. Key features include AI-powered resume review with PDF parsing, interactive learning modules, real-time group chat functionality, and secure user authentication. The app implements modern Flutter architecture with Provider state management, secure API key storage, offline mode support, and Firebase Cloud Messaging for push notifications. Built with a focus on user experience, the app includes dark mode support, advanced search capabilities, and a clean, intuitive UI design.
